Введение:
Современные компьютерные сети передают огромные объемы данных, и анализ сетевого трафика является важной задачей для администраторов, разработчиков и специалистов по информационной безопасности. Снифферы (анализаторы трафика) позволяют перехватывать и анализировать сетевые пакеты, что помогает в диагностике сетевых проблем, выявлении аномалий и исследовании протоколов.

В данной лабораторной работе реализован простой анализатор сетевого трафика, способный перехватывать и разбирать IP-, TCP-, UDP- и DNS-пакеты. Программа использует raw-сокеты для захвата трафика и предоставляет удобный интерфейс для просмотра структуры пакетов.	
Цель:
Разработка программы для захвата и анализа сетевого трафика с возможностью парсинга IP-, TCP-, UDP- и DNS-заголовков.
Задачи:
Реализовать захват сетевых пакетов с использованием raw-сокета.

Привязать сокет к выбранному IP-адресу.

Настроить параметры сокета для захвата всех входящих и исходящих пакетов (IOControlCode.ReceiveAll).

Реализовать парсинг IP-заголовков, включая версию, длину, TTL, протокол, адреса источника и назначения.

Реализовать парсинг TCP-заголовков (порты, флаги, контрольные суммы).

Реализовать парсинг UDP-заголовков (порты, длина пакета).

Реализовать парсинг DNS-запросов и ответов (транзакции, флаги, типы записей).

Организовать вывод информации в удобном для анализа виде (дерево пакетов).
